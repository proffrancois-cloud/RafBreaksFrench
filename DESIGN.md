# Design

## North Star

FrenchEase should feel like a serious Francophone education workspace: crisp, calm, and purpose-built for IB preparation. The interface is a tool, not a brochure. It should reduce cognitive load by showing one decision at a time and keeping complex question banks behind clear, predictable disclosure.

## Color System

Use the Organisation internationale de la Francophonie color family coherently:

- OIF blue `#00ADEF`: primary action, selected navigation, current learning focus.
- OIF green `#78CA2A`: success, mastery, sustainable/progress cues.
- OIF yellow `#FCD205`: warning, needs review, planning emphasis.
- OIF red `#EE2631`: error, danger, incorrect feedback.
- OIF purple `#672290`: creativity, culture, secondary theme identity.

For accessibility, full-saturation colors are not used as small body text on white. Each color has darker ink and softer background variants. Neutrals carry most of the UI.

## Typography

Use one professional sans-serif stack. Product screens use fixed type sizes, strong weight contrast, and restrained scale jumps. Large display typography is reserved for actual dashboard status, not every page.

## Layout

Desktop uses a persistent side navigation and a structured content area. Mobile collapses structurally into a sticky select navigation. Panels use 8px radius or less, light borders, and very subtle elevation. Cards are for real objects: tasks, texts, audio, practice panels, and progress rows.

## Interaction

Buttons, tabs, selectors, forms, and disclosure panels share one visual vocabulary. Primary actions are blue. Secondary actions are neutral. Focus states are visible. Question banks, transcripts, and writing workspaces use progressive disclosure so the first view remains simple.

## Research Notes

OIF mission language emphasizes French language, cultural and linguistic diversity, education/training/research, democracy/human rights, and sustainable development. The UI should therefore feel multilingual, culturally serious, and education-first. UX research on cognitive load and recognition over recall supports reducing visible choices, grouping related decisions, and keeping contextual help near the task.
